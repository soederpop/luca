# Luca Visual Representations

I want to build a UI where I can chat with Luca and not just an image, maybe an animation of e.g. a current mood or him thinking, typing, expressing an emotion

It isn't feasible to use stable diffusion live to do this, but we could pregenerate a wide range of options by running a bunch of comfy ui workflows. We can use Runpod to speed up generation as ComfyUI on the Mac kind of slow.
