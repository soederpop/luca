# ContentBase Writing Assistant

Contentbase works by defining models, which are expecting underlying markdown files to adhere to certain heading structures and have certain yaml frontmatter (optional), and to have some sort of sensible organization with good names when it comes to folders.

We should be able to prompt an AI with all of this information about the whole project of writing and have it help us:

- with the overall writing process ( based on the types of stuff we're writing )
- writing an individual type of document within the project

## Motivation

- To reduce the overhead for the author having to conform to certain structure
- To use AI to help write in a structured way
