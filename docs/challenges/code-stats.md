# Code Stats Report

Build `luca code-stats [path]` (defaulting to the current project), using only the features available in luca. It should report:

- file counts grouped by extension
- all TODO / FIXME / HACK annotations with file and line
- the 5 largest files

Output a readable colored report by default, and machine-readable output with `--json` (the JSON must be clean — no decorative output mixed in).

## After you are done

Write a LESSONS.md in the project root that describes what you learned, what you struggled with, and what you could have been supplied with up front either in the CLAUDE.md or in the skills that come with luca so you could achieve the goal quicker and with less trouble.
